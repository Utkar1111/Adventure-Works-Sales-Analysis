# Adventure Works Sales Analysis

# Project Overview
This project focuses on analyzing Adventure Works sales data to identify business trends, customer behavior, product performance, and regional sales insights. The analysis was performed using Excel, SQL, Power BI, and Tableau to support data-driven decision-making.

---

## Objective
- Analyze sales performance across different regions.
- Identify top-performing products and categories.
- Evaluate customer purchasing behavior.
- Track important KPIs such as Sales, Profit, Orders, and Quantity.
- Create interactive dashboards for business insights.

---

## Tools & Technologies
- Microsoft Excel
- SQL
- Power BI
- Tableau

---

## Dataset
Adventure Works Sales Dataset

---

## Dashboard Features
- Sales Overview
- Profit Analysis
- Customer Analysis
- Product Performance
- Regional Performance
- KPI Dashboard

---

## Key Insights
- Identified top-selling products.
- Compared regional sales performance.
- Tracked monthly sales trends.
- Analyzed customer contribution.
- Improved business reporting using interactive dashboards.
